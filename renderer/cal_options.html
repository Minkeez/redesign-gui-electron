fix intrinsic
Fix the values of fx, fy, cx, and cy based on the initial guess provided to OpenCV by the DICe calibration tool.
use intrinsic guess
Begin with the initial values of fx, fy, cx, and cy provided to OpenCV then optimize further. Otherwise, (cx, cy) is initially set to the image center, and focal distances are computed in a least-squares fashion.
use extrinsic guess
Begin with the initial values of the extrinsic parameters and optimize further.
fix principal point
Use the image centroid as the principal point.
fix aspect ratio
Only consider fy a free parameter, the ratio between fx and fy is fixed.
same focal length
Constrain fx and fy to be the same.
zero tangent dist
Set tangential distortions to zero.
fix K1
Keep the K1 parameter constant during the optimization. If use_initial_guess is not active, K1 is set to 0. (Same for K2 and K3 if selected below)
